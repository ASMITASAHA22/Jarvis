# Installation

1. Clone the Repository

        https://github.com/ASMITASAHA22/Jarvis.git

2. Open the Project Directory

        cd Jarvis-AI-using-python3-

3. Install the Required Python Modules

        pip install -r requirements.txt

4. Run the Script

        python jarvis.py
